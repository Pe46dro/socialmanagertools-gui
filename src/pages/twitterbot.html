    Twitter nickname without @

    Your password is stored locally

    Bot go to random hashtag from list, like 19-20 tweet fast and stop X minutes in loop

    10-15 hashtag is raccomanded. Separate with comma, example: #muraleshunter, picoftheday, selfie

  • Max like/retweet for day

    Twitter limit is 1000/day, recommended value is 900

  • Language of tweets

    Filter tweets language

  • 2FA Pin (BETA)

    If you received sms or email pin, write it and wait

  • Google Chrome Path

    Install google chrome, this bot work with google chrome api

    If bot don't work, enable this mode
